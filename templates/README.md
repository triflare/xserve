# Extension Template Gallery

Mint includes starter templates for common extension patterns.

## Available templates

- `blank` - Empty extension starter
- `sensing-blocks` - Reading Scratch runtime state
- `operators` - Math and string operations
- `control-flow` - Custom control blocks and flow helpers
- `looks` - Visual effects and rendering helpers
- `data-storage` - Variables and list-style state
- `advanced` - Multi-module extension with shared services

Run `npm run init` and choose a template to scaffold `src/`.
